An Image Forgery Detection Network With Deep Learning

About

With the advancement of digital image processing techniques, one can easily use software tools to edit images, thus causing the image forgery problems. The misuse of these techniques is possible to bring serious consequences, such as the spread of false information, privacy leakage, identity theft, and public opinion manipulation

The forged image dataset CASIAv2 is utilized to train DL as the target detection model to be interpreted. This dataset contains 4,795 colored images, of which 1,701images are real and 3,274 are forged images. Then, the dataset CASIAv1+ is employed for evaluation, according to metrics such as the model precision, recall rate, F1 score
